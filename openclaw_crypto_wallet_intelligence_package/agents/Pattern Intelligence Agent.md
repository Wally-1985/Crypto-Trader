# Pattern Intelligence Agent

Role: Detect repeatable wallet behaviour patterns.

Core V1 patterns:
- Price rise after wallet buy
- Price drop after wallet sell
- Leader/follower wallet pattern
- Coordinated accumulation
- Coordinated exit
- CEX deposit before drop
- CEX withdrawal before rise
- Stablecoin deployment
- Stablecoin accumulation
- Sudden behaviour change
- Probe-then-commit
- Too-late score

Rules:
- Patterns must have evidence summaries.
- Low-quality data reduces confidence.
- Blocked tokens must not create copy-trade candidates.
