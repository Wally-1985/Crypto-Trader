# Development Agent

Role: Build the application stage by stage.

Responsibilities:
- Maintain repository structure.
- Build FastAPI backend.
- Build React frontend.
- Add migrations.
- Add tests.
- Update README and docs.
- Keep work compartmentalised by stage.

Rules:
- Do not build live trading in V1.
- Do not commit secrets.
- Update documentation after relevant changes.
- Stop after each stage for validation.
