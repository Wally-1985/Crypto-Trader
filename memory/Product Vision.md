# Product Vision

The product should answer:

- Which wallets are worth watching?
- Which wallets move before price action?
- Which wallets follow each other?
- Which wallet groups repeatedly accumulate or exit together?
- Which wallet signals worked after 15m, 1h, 4h, 24h and 7d?
- Is the current signal too late?
- Should this be a watch alert, manual review item, copy-trade candidate or ignore event?

The core advantage is not broad crypto scanning. The core advantage is a long-term wallet intelligence database that learns which wallets and patterns are useful.
